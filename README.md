# オズボーン・アイデア拡張GPT

オズボーンのチェックリスト（9分類）を使ってアイデアを多角的に拡張する、初心者向けカスタムGPTです。

このカスタムGPTは、アレックス・F・オズボーンが開発した**オズボーンのチェックリスト**を活用し、ユーザーのアイデアや課題を**9つの視点**から体系的に拡張します。初心者でも分かりやすく、実践的なアイディエーションを通じてオズボーンのチェックリストの考え方を体験できます。

## 🎯 主な機能

- 💡 **新しいアイデアの多角的な拡張**: 既存のアイデアを9つの視点からブラッシュアップ
- 🔧 **製品やサービスの改良提案**: 転用・応用・変更などの観点から具体的な改良策を提示
- 📊 **ミニ評価機能**: 各提案に目的適合度・実現容易性・差別化を★で評価
- 🎨 **創造的な発想のヒント**: 行き詰まったときの新しい視点を提供
- 📝 **実装ヒントとリスク提示**: 各提案に短い根拠・実装ヒント・リスクを添付

## 🧩 オズボーンのチェックリスト（9分類）

1. **転用（Put to other uses）** - 他の用途は？他の分野で使える？
2. **応用（Adapt/Adjust）** - 類似事例から借りられる仕組みは？
3. **変更（Modify）** - 仕様・UI・体験を変えたら?
4. **拡大（Magnify）** - 機能・規模・精度を強化すると？
5. **縮小（Minify/Eliminate）** - 小型化・簡略化で価値は残る？
6. **代用（Substitute）** - 素材・工程・人を置き換える？
7. **再配置（Rearrange）** - 順番・責任分担・レイアウトを入れ替えると？
8. **逆転（Reverse）** - 流れ・料金・主従を反転すると？
9. **結合（Combine）** - 他プロダクト・機能を組み合わせると？

## 🚀 使い方

### セットアップ

1. [ChatGPT](https://chat.openai.com/)（ChatGPT Plus/Pro/Team/Enterprise）にログイン
2. 左サイドバーから「GPT」→「調べる」を選択
3. 右上の「＋ 作成する」をクリック
4. 「構成」タブを開く
5. このリポジトリのファイルを以下のように設定：

#### 基本設定

- **Name**: 
  ```
  オズボーン・アイデア拡張GPT
  ```

- **Description**:
  ```
  入力されたアイデアを「オズボーンのチェックリスト（9分類）」で多角的に拡張します。初心者でも使えるよう、各提案に短い根拠・実装ヒント・リスクも添えます。
  ```

- **Instructions**: 
  `instructions.md` の内容をコピー&ペースト

- **Conversation starters**: 
  `conversation-starters.txt` から選択

- **Knowledge**: 
  `knowledge/` フォルダ内のファイルをアップロード
  - `osborn_checklist_ja.md`

#### 機能設定

- ✅ **Web Browsing**: ON（最新事例の検索に使用）
- ✅ **Canvas**: ON（資料作成機能）
- ❌ **DALL·E Image Generation**: OFF
- ✅ **Code Interpreter**: ON（資料分析機能）

### 実際の使い方

1. **まずは自由に話してみる**: 「こんなアイデアを考えているんだけど...」と気軽に話しかけてください
2. **課題を明確にする**: 「〇〇の継続率を上げたい」「△△を改善したい」など具体的なテーマを伝えると効果的
3. **提案された改良案を評価する**: 気に入った案があれば、さらに深掘りをリクエスト
4. **オズボーンのチェックリストの学習も兼ねる**: 使われた分類を意識することで、自然と発想法が身につきます
